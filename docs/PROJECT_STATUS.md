# Project Status (Public Repo)

This document tracks the public frontend UI and documentation. Core backend services, infrastructure, and operational details are maintained in the private repository.

## Current focus

- Improve the landing page and public-facing UI.
- Build reusable UI components and layouts.
- Expand documentation and onboarding for contributors.
- Add accessibility and responsiveness improvements.

## Known limitations

- The backend API is private, so some screens require mocked data or the live API.
- Environment variables in this repo are placeholders only.

## How to help

- Pick an issue labeled for GSSOC 2026 or a task from [BEGINNER_TASKS.md](../BEGINNER_TASKS.md).
- Submit focused PRs with screenshots for UI changes.

## Private backlog

Backend technical debt, deployments, and infrastructure tasks are tracked privately and are not listed here.
